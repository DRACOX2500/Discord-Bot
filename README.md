## Requirement

- nodejs 16+
- npm 9+
- ffmpeg
    - Linux
        1. Update the repository `sudo apt update && sudo apt upgrade`
        2. Install FFmpeg `sudo apt install ffmpeg`
        3. Verify the installation `ffmpeg -version`

## Installation

1. clone project: ``
2. npm i
3. config env + bot config
4. build & start
